# DigiKaksha
To see the working oif the project kindly follow this YoutTube link: 
This is an ML based Smart solution to the Educational institutes. It has Mainly 3 components:

a) Face detection made using OpenCV
b) Graphs for attendance Management
c) Langchain based chatbot for Helping students in understanding their notes

The tech / modules or libraries used in it are:

i) Langchain and RAG
ii) OpenCV
iii) Matplotlib
iv) MongoDB (For Database)

Some points to keep in Mind while working with this project :

1) You have to do pip install requirements.txt to install the needed modules.
2) You need an API key for :
                    i) LANGCHAIN_TRACING_V2 = ""(Langchain official API key)
                    ii) GOOGLE_API_KEY = "" (GEMINI API Key)

3) A proper Camera Setup , it can be an inbuilt camera  or an external secondary camera.
