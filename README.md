# DigiKaksha
To see the working of the project kindly follow this YoutTube link: https://youtu.be/s1tvwx_CCRs

This is an ML based Smart solution for assisting and managing all the taks in the Educational institutes. It has Mainly 3 components:

a) Face detection based Attendance monitoring made using OpenCV
b) Graphs for analysis of the student's perofrmance
c) Langchain based chatbot for Helping students in understanding their notes or Study Material

The tech / modules or libraries used in it are:

i) Langchain and RAG
ii) OpenCV
iii) Matplotlib
iv) MongoDB (For Database)

Some points to keep in Mind while working with this project :

1) You have to do pip install requirements.txt to install the needed modules.
2) Kindly use Python 3.11.X or versions below it. It may face problems in Python 3.12.X
3) You need an API key for :
                    i) LANGCHAIN_TRACING_V2 = ""(Langchain official API key)
                    ii) GOOGLE_API_KEY = "" (GEMINI API Key)

4) A proper Camera Setup , it can be an inbuilt camera  or an external secondary camera.
